Description

JAME is an application for creating fractals and other graphics artifacts. JAME has a real-time, multi-threaded graphics engine with layers, filters, effects, a lot of formulas, and parameters. JAME supports operations such as continuous zoom of Mandelbrot and Julia sets, rotation and color shifting. JAME has also advanced features such as distributed computation and scripting. You can take photos of your fractals or record movies from your exploration paths. Photos and movies are stored in a database to be rendered in high resolution and exported to a file. JAME can be extended by plug-ins which provide fractals, formulas, filters, effects and more.

How to install

Unzip the archive in your preferred location. JAME requires Java JRE 8 or later. You can download the latest Java JRE from Oracle.

How to run

For Mac OS X click on the app. For Windows click on JAME.bat. For Linux run script JAME.sh.

How to distribute computation

JAME supports distributed computation based on the JXTA peer-to-peer technology. To enable distributed computation, open the ports from 9701 to 9704 in your firewall and select the JXTASpool in jobs tab of service window of JAME. When you select the JXTA spool, JAME automatically discovers other instances of JAME on the local network, and distribute to these peers the jobs while rendering the clips.

How to build

Download Maven and Ant. Make sure mvn and ant commands are in your system path. Define a M2_HOME environment variable equals to the Maven path. Checkout JAME source code from SourceForge.

Open a terminal, change current folder to maven/net.sf.jame.build and execute the command:

ant all
